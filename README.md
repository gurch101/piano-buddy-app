# Piano Buddy App

- nestjs backend, mithriljs frontend.
- Uses npm workspaces. Add a dependency by running `npm install <dep> -w <package>`.

### Setup

##### Development

```sh
npm install
cd client && npm dev # runs on port 3000
cd server && npm dev # runs on port 8080
```

##### Production

```sh
npm install
npm run build
npm start
```
